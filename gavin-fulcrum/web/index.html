Connect Electron Cash
  1. Wait for the first index build to finish (this takes a while).
  2. In Electron Cash: Network → Server, turn off auto-connect.
  3. Add umbrel.local port 50022 (SSL) and select it.
  4. Accept the self-signed certificate — it's your own server.
